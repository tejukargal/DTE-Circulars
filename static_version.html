DTE Karnataka Circulars

Latest Government Orders, Circulars & Notices

Note: This is a Python application that scrapes live data from the DTE website. To use it, you need to download and run it on your computer.

🚀 How to Use:

  1. Download and extract the ZIP file
  2. Make sure you have Python installed on your computer
  3. Open command prompt in the extracted folder
  4. Run: pip install -r requirements.txt
  5. Double-click START_HERE.bat to launch the app
  6. Choose your preferred interface (Desktop, Web, or Command Line)

📱 Available Interfaces:

  • Desktop App: Modern GUI with clickable links
  • Web App: Browser-based interface (localhost:5000)
  • Command Line: Terminal-based scraper

✨ Features:

  • Fetches latest 10 circulars from DTE Karnataka
  • Clickable PDF links
  • Export to JSON
  • Real-time refresh
  • Handles Kannada text properly
